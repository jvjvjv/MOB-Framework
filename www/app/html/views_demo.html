• Views

    Views consist each of an HTML, CSS, and JS file. For example, for this view, called "views_demo" there is a corresponding views_demo.html, views_demo.css, and views_demo.css.

    The slide-out menu includes an option to display viewHistory. Tapping on it will reveal each View ID saved in history.

  • MOB.views.go()

    Go to Login View (save in history)

    Do not save in history can be tested by signing in, and then tapping on the Back button on the Home View. You should return back to this page, as the Login View was not saved in the View History.

    Go to Login View (do not save in history)

  • MOB.views.goBack()

    Go back to previous View

  • MOB.views.goBackTo()

    Go all the way back to Home View

  • MOB.views.template()

    Add a templated item

  • MOB.views.menuToggle()

    open the menu.
